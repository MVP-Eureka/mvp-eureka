#front-end

O arquivo está sem as dependencias do node, segue o tutorial de como instalar:
<-------------------------- Escreva no terminal -------------------------------->
- cd client
- npm i
- npm run dev
<----------------- Assim suas dependencias vão ser baixadas!!! ----------------->

Projeto utilizando tailwind para estilização, então, fatores importantes:
instalar a extensão Tailwind CSS IntelliSense no VS
instalar a extensão Prettier - Code formatter no VS

para organização do codigo:
<------------------ Execute os seguintes comandos no terminal ------------------>

- npm install tailwindcss @tailwindcss/vite
- npm install -D prettier prettier-plugin-tailwindcss

eles vão baixar o tailwind e uma funcionalidade da extensão Prettier - CF 
de formatação automatica das classes no tailwind, respectivamente
